####列表
列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加1. 2. 3. 符号要和文字之间加上一个字符的空格。

- 桃子
- 西瓜

####引用
只需要在文本前加入 > 这种尖括号（大于号）即可
>不知道引用谁的话

####粗体和斜体
Markdown 的粗体和斜体也非常简单，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。
*这是斜体*  **这是粗体**

####代码框
在 Markdown下实现也非常简单，只需要用两个 ` 把中间的代码包裹起来。

`
int main(){
    cout << "hello world " << endl;
    return 0;
}
`
